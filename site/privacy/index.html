개인정보처리방침

최종 업데이트: 2026년 4월 12일

1. 수집하는 정보

KnitMate는 다음과 같은 정보를 수집합니다.

  • 계정 정보: 이메일 주소, 비밀번호(암호화 저장), 사용자명, 자기소개
  • 소셜 로그인: Google 또는 Apple 계정을 통한 OAuth 인증 시 해당 제공자가 전달하는 기본 프로필 정보
  • 업로드 콘텐츠: 프로필 사진, 피드 게시물 사진, 뜨개 스냅샷 이미지
  • 커뮤니티 활동: 피드 게시물, 뜨모저모 게시물, 댓글
  • 뜨개 작업 데이터: 프로젝트 제목, 단수 기록, 메모, 히스토리 — 기기 로컬 저장소에만 보관

2. 정보 이용 방식

  • 계정 생성·로그인 및 본인 확인
  • 피드·커뮤니티 기능 제공(게시물, 댓글, 프로필 표시)
  • 업로드된 사진 및 스냅샷 저장과 표시
  • 앱 오류 파악 및 서비스 개선

3. 제3자 서비스

KnitMate는 인증·데이터베이스·파일 저장에 Supabase(Supabase Inc., 미국)를 사용합니다. Supabase는 계정 정보, 게시물, 댓글, 업로드 이미지를 처리하며, 자체 개인정보보호 정책에 따라 데이터를 관리합니다. Google 또는 Apple 소셜 로그인을 사용하는 경우 해당 제공자의 개인정보처리방침도 적용됩니다.

4. 사진 접근 권한

앱은 프로필 사진 설정 및 피드·뜨모저모 게시물 작성 시 기기 사진 보관함에 접근합니다. 카메라 직접 촬영 기능은 제공하지 않으며, 사진 보관함 접근은 사용자가 직접 선택한 경우에만 이루어집니다.

5. 데이터 보관 및 삭제

  • 뜨개 작업 데이터(단수, 메모 등)는 기기 로컬 저장소에만 저장되며, 앱 삭제 시 함께 삭제됩니다.
  • 계정 및 소셜 데이터(게시물, 프로필, 업로드 파일)는 Supabase 서버에 보관됩니다.
  • 계정 삭제를 원하시면 앱 설정 화면 또는 아래 이메일로 요청하시면 처리해 드립니다.

6. 외부 공유

KnitMate는 사용자의 개인정보를 광고 목적으로 판매하거나 제3자에게 무단으로 공유하지 않습니다. 서비스 운영에 필요한 Supabase 외에는 개인정보를 전달하는 외부 서비스가 없습니다.

1. 수집하는 정보

KnitMate 현재 버전은 사용자 계정, 서버 저장, 광고 추적 기능을 제공하지 않습니다. 프로젝트 제목, 메모, 단수 기록, 히스토리 데이터는 사용자의 기기 로컬 저장소에만 저장됩니다.

2. 정보 이용 방식

로컬에 저장된 데이터는 앱 기능 제공을 위해서만 사용됩니다. 현재 버전에서는 이 데이터가 개발자 서버나 제3자 서비스로 전송되지 않습니다.

3. 외부 공유

KnitMate는 현재 버전에서 사용자의 프로젝트 데이터나 식별 정보를 외부에 판매하거나 공유하지 않습니다.

4. 데이터 보관

데이터는 앱이 설치된 기기의 로컬 저장소에 유지됩니다. 앱을 삭제하거나 기기 데이터를 초기화하는 경우 저장된 프로젝트 정보가 삭제될 수 있습니다.

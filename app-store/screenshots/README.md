# Screenshot Plan

App Store용 실제 스크린샷을 만들 때 아래 4장을 우선 준비하면 MVP 설명력이 가장 좋습니다.

1. 홈 화면
   최근 작업 프로젝트와 진행 중 프로젝트 목록이 보이도록 구성
2. 프로젝트 상세
   현재 단수, 최근 메모, 최근 수정 시간이 보이도록 구성
3. 작업 화면
   큰 단수 숫자와 `+1 / -1`, 자동 저장 상태가 함께 보이도록 구성
4. 히스토리 화면
   변경 이력 목록과 복구 버튼이 보이도록 구성

추천 캡션:

- "지금 몇 단인지 잃지 않도록"
- "프로젝트별 단수와 메모 관리"
- "조용한 자동 저장"
- "원할 때 이전 상태로 복구"
